1
Google Search Process

Sabse pehle aapko Google par search karna hai. Browser open karo aur search bar mein ye exact words type karo:

"Free Rdp Expert server Rdphosting"

Search results mein RdpHostings.com Expert server wala website ka link dikhega. Is official website pe click karo. Ye trusted aur reliable service hai jo free RDP provide karti hai.


2
Website Navigation

RdpHostings website open hone ke baad, homepage par carefully dekhiye. Yahan aapko different sections dikhenge.

Main page par "Expert server" ka section dhundiye. Ye section usually homepage par prominently display hota hai. Isme free RDP server ke options milte hain. Agar Expert wala nhi show krta aur uski jagah Solo server Show krta h to tu glt jgh pr h

Expert server section mein jakar signup area locate karo.

3
Signup Process - Important

Expert server area mein "Sign up" button par click karo. Ek naya page khulega jahan signup form dikhega.

Dhyan rakho: Jab signup page khule, to aapko Category aur Action ke dropdown options dikhenge. In dropdown options par bilkul click mat karo! Ye bohot important hai.

In dropdown menus ko completely ignore karo aur direct "Hostname" field mein jao.

Hostname field mein exactly ye type karo: Nobitardp

Ye hostname fill karne ke baad neeche "Continue" button par click karo.

4
Configuration Settings

Continue button click karne ke baad configuration page open hoga. Is page par aapko teen (3) checkboxes dikhenge.

Ye teen checkboxes server ke specifications aur features ke liye hain. Sabko tick karna zaroori hai:

‚úÖ Pehla checkbox tick karo

‚úÖ Doosra checkbox tick karo

‚úÖ Teesra checkbox tick karo

Teeno checkboxes select karne ke baad "Checkout" button par click karo.

5
Account Details Fill Karna

Ab aapke saamne account creation form aayega. Sari details carefully aur sahi fill karo:

Full Name: Apna complete real name likho

Email Address: Working email ID daalo jo aap regularly check karte ho

Phone Number: Apna real mobile number daalo (verification ke liye zaroori hai)

Address: Complete address fill karo

Password: Strong password create karo aur yaad rakhne wala

Phone number bilkul real daalo kyunki verification ke time jarurat pad sakti hai. Fake number mat daalo.

Sab kuch fill karne ke baad "Complete Order" par click karo.

6
Order Confirmation & Email Check

Order complete hone ke baad aapko screen par Order Number dikhega. Is number ka screenshot le lo backup ke liye.

Ab apna email check karo. RdpHostings se 2-3 emails aayenge. Email check karne ka proper tareeka:

Step 1: Primary inbox check karo

Step 2: Agar nahi mila to Social tab check karo

Step 3: Promotions tab bhi dekho

Step 4: Last mein Spam/Junk folder check karo

Email mil jayega, bas thoda wait karna pad sakta hai.

7
Email Verification Process

Emails mein se "Email Confirmation" wala email dhundho. Ye verification ke liye important hai.

Email open karo aur andar confirmation link dikhega. Is link ko handle karne ke do tareeke hain:

Method 1: Direct link par click karo. Agar browser mein open ho jaye to perfect.

Method 2: Agar link click se nahi khul raha, to link ko copy karo aur naye browser tab mein paste karke open karo.

Successful verification ke baad ye message dikhega: "Your email address has been verified"

Verification success page par "Continue to Client Area" button par click karo.


8
Client Area Mein Server Status

Client area dashboard open ‡§π‡•ã‡§®‡•á ke baad "View All" button dhundho aur click karo.

Services page open hoga jahan aapka Expert server dikhega. Abhi server status "Pending" show kar raha hoga.

Pending ka matlab hai server setup ho raha hai. Ye process 5 se 30 minute tak le sakti hai.

Patience rakho aur page ko bar-bar refresh mat karo. Server ready hone par automatically email aa jayega.

9
Server Ready Email
NOTE: RDP READY wala email 24 hours ke andar ‡§π‡•Ä ‡§Ü‡§§‡§æ ‡§π‡•à ‚Äì patience rakh, baar baar refresh karke Pdeshaan nhi hona hai!

"Your RDP Server Is Ready To Login!"

Ye email bohot important hai kyunki isme aapki server details hongi:

IP Address: Server ka unique address

Username: Login karne ke liye username

Is puri email ka screenshot le lo aur safe jagah save karo. Ye details future mein kaam aayengi.

IP address aur username ko separately bhi note kar lo.


10
Password Retrieve Karna

Server ready hone ke baad password get karne ke liye is specific link par jao:

https://rdphostings.com/rdp/clientarea.php?action=services

Ye link copy karo aur browser mein paste karke open karo. Agar login page aaye to apne account details se login karo.

Services page mein apna server dhundho aur click karo. Server details page mein ye milega:

IP Address: Confirm karo ki ye email wala same hai

Current Password: Ye daily change hota hai

Password roz change hota hai! Har time connect karne se pehle yahan se current password check karo.

Password carefully copy karo aur safely save karo.

Congratulations! Aapka free RDP server completely ready hai. Connection process ke liye next tutorial follow karo.
11
Windows Remote Desktop App

RDP server connect karne ke liye app install karna zaroori hai. App store (Play Store ya App Store) open karo.

Search mein type karo: "Microsoft Remote Desktop"

Official Microsoft app dhundho. Ye free app hai aur Microsoft ka original app hai.

Install button par click karo aur app download hone ka wait karo.

App install ho jayega but abhi open mat karo. Setup process complete ho gaya hai.
